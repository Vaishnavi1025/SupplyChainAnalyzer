# SupplyChainAnalyzer

## Project Overview
Java-based Supply Chain Analyzer that processes CSV data and outputs an HTML dashboard with Avg Availability, Total Sold, and Avg Revenue KPIs.

**Phase 1:** Console-based Java program to read 101-row supply chain data  
**Phase 2:** Upgraded to generate interactive `dashboard.html` for business users

## Tech Stack
`Java` `OOP` `File I/O` `HashMap` `HTML` `CSS`

## Files in this Repo
1. `DashboardAnalyzer.java` - Backend: Reads CSV and calculates KPIs
2. `dashboard.html` - Frontend: Interactive dashboard for business decisions  
3. `SupplyChain.png` - Screenshot of the dashboard output

## How to Run
```bash
javac DashboardAnalyzer.java
java DashboardAnalyzer
